# Расчет коэффициента размножения нейтронов в гомогенном реакторе

Данный репозиторий содержит Jupyter Notebook с лабораторной работой по физике ядерных реакторов. В работе проводится расчет эффективного коэффициента размножения нейтронов ($k_\infty$) для гомогенной активной зоны с использованием **формулы четырех сомножителей**.

## Содержание

*   **Цель работы:** Исследовать зависимость коэффициента размножения нейтронов от обогащения урана и состава материалов активной зоны для реакторов типа РБМК и ВВЭР.
*   **Методика:** Расчет базируется на формуле $k_\infty = \mu \phi \theta \eta$, где:
    *   $\mu$ — коэффициент размножения на быстрых нейтронах.
    *   $\phi$ — вероятность избежать резонансного захвата.
    *   $\theta$ — коэффициент использования тепловых нейтронов.
    *   $\eta$ — число нейтронов деления на один поглощенный в топливе тепловой нейтрон.
*   **Инструменты:** Код написан на Python с использованием библиотек `pandas`, `numpy`, `matplotlib` и `mendeleev`.

## Результаты

В ходе работы были построены графики зависимости коэффициента размножения от обогащения урана для различных комбинаций материалов (топливо, теплоноситель, замедлитель, конструкционные материалы) для реакторов РБМК и ВВЭР.

**Основные выводы:**

*   Для реактора **РБМК** наиболее подходящими оказались комбинации с тяжелой водой (`D2O`) в качестве теплоносителя/замедлителя и железом (`Fe`) в качестве конструкционного материала.
*   Для реактора **ВВЭР** оптимальные значения коэффициента размножения достигаются при использовании обычной воды (`H2O`) и циркония (`Zr`).
*   Исследована зависимость вероятности избежать резонансного захвата ($\phi$) и коэффициента использования тепловых нейтронов ($\theta$) от объемных долей компонент.

Более подробные результаты и графики представлены в самом [notebook](neutron_multiplication_factor.ipynb).
